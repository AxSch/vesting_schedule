# Vesting Schedule

## Requirements
* python 3.13
* pipenv
* A strong will :)


## Setup

1. Setup & activate the project's virtual env
```shell
pipenv install
```

```shell
pipenv shell
```

2. Run the program
```shell
pipenv run vesting_schedule [csv_file] [target_date] [precision]
```

3. Run the tests
```shell
pytest . 
```

3. Enjoy the sunrise :')
